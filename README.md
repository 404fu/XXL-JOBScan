# 描述：

**这是一款多线程批量探测XXL-JOB默认口令的脚本工具**

# 使用教程：

- url.txt文件内为目标地址
- **启动命令1（批量扫描）python3 weakpass-xxljob.py -f url.txt -o res.txt -n 30**
- 启动命令2（单独扫描）python3 weakpass-xxljob.py -u http://target.com
